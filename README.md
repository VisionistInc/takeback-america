# HHS Code-a-thon

## Running the app:

### With Docker:

```bash
git clone git@github.com:VisionistInc/takeback-america.git
cd takeback-america
make deps
make docker-build && make docker-run
```

### With Node:

```bash
git clone git@github.com:VisionistInc/takeback-america.git
cd takeback-america
yarn
yarn start
```
